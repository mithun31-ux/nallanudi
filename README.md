NallaNudi is an Android application developed using Kotlin and Jetpack Compose with the support of Generative AI concepts.
The application is designed with a clean UI/UX approach and focuses on delivering a smooth and user-friendly mobile experience.

🚀 Features

* Modern Android UI using Jetpack Compose
* Kotlin-based application architecture
* Smooth navigation and responsive design
* Firebase integration support
* AI-assisted development workflow
* Optimized user experience
* Scalable and maintainable code structure

🛠️ Tech Stack

* Kotlin
* Jetpack Compose
* Android Studio
* Firebase
* Google AI Studio
* Material Design 3

Installation

1. Clone the repository:
2. git clone https://github.com/Mithun31-ux/NallaNudi.git

3. 2. Open the project in Android Studio
3. Sync Gradle files
4. Run the application on Emulator or Android Device
